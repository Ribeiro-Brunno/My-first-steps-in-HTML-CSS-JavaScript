Resumo

(Voltar)

HTML é uma linguagem de marcação que é usada para criar páginas da web. Ela é composta por tags, que são símbolos que definem o conteúdo e o formato da página. HTML é a base para criar sites e outros documentos HTML. Sem HTML, não seria possível criar páginas da web. HTML é responsável por definir a estrutura, o conteúdo e o formato das páginas da web. HTML é uma habilidade valiosa para o mercado de trabalho. HTML é uma habilidade essencial para qualquer pessoa que queira trabalhar com desenvolvimento web. Empresas de todos os tamanhos estão procurando por profissionais que saibam HTML.

Ferramentas utilizadas

(Voltar)

Como usar o HTML no Visual Studio Code usando a extensão Live Server

Para usar o HTML no Visual Studio Code usando a extensão Live Server, siga estas etapas:

  1. Instale a extensão Live Server.

    2. Para instalar a extensão Live Server, abra o Visual Studio Code e clique em Extensões. Na barra de pesquisa, digite "Live Server" e clique em Instalar.

  2. Crie um novo arquivo HTML.

    3. Para criar um novo arquivo HTML, clique em Arquivo > Novo > Arquivo HTML.

  3. Insira o código HTML no arquivo.

    4. Insira o código HTML que você deseja usar no arquivo.

  4. Salve o arquivo.

    5. Para salvar o arquivo, clique em Arquivo > Salvar.

  5. Pressione F5 ou clique no botão Play no canto superior direito do Visual Studio Code.

    6. Pressionar F5 ou clicar no botão Play abrirá um servidor local e abrirá o arquivo HTML no navegador padrão.

    A extensão Live Server abrirá um servidor local na porta 5500. Você pode alterar a porta do servidor local nas configurações da extensão Live Server.

    Você pode usar a extensão Live Server para visualizar seus arquivos HTML em tempo real. Isso é útil para depurar seu código e ver como ele aparece no navegador.

Estruturas básica do HTML

(Voltar)
  • Cabeçalho (head): O cabeçalho é a parte do HTML que contém informações sobre o documento, como o título, as meta tags e os scripts.
  • Corpo (body): O corpo é a parte do HTML que contém o conteúdo do documento, como texto, imagens, vídeos e outros elementos.
  • Elementos (tags): Os elementos são as unidades básicas do HTML. Eles são usados para definir o conteúdo e o formato do documento.
  • Atributos (attributes): Os atributos são usados para especificar propriedades dos elementos.

As estruturas básicas do HTML são essenciais para a criação de documentos HTML. Ao aprender sobre essas estruturas, você poderá criar sites e outros documentos HTML personalizados.

Falando sobre Tags

(Voltar)

As tags do HTML são os elementos básicos que compõem um documento HTML. Elas são usadas para definir o conteúdo e o formato do documento.

As tags são compostas por dois símbolos: um abre-tag e um fecha-tag. O abre-tag indica o início do elemento, e o fecha-tag indica o fim do elemento.

Entre os abre-tags e fecha-tags, é possível inserir conteúdo, como texto, imagens, vídeos e outros elementos.

Atributos das Tags

(Voltar)

As tags também podem conter atributos. Os atributos são usados para especificar propriedades do elemento

Alguns exemplos de tags do HTML são:

  • < html> e < /html> - definem o início e o fim do documento HTML.
  • < head> e < /head> - definem o cabeçalho do documento HTML.
  • < title> e < /title> - definem o título do documento HTML.
  • < body> e < /body> - definem o corpo do documento HTML.
  • < h1>, < h2>, < h3>, < h4>, < h5>, < h6> - definem cabeçalhos de diferentes níveis.
  • < p> - define um parágrafo.
  • < img> - insere uma imagem no documento HTML.
  • < a> - cria um link para outro documento ou recurso.

As tags do HTML podem ser agrupadas em diferentes categorias, de acordo com sua função. As principais categorias de tags são:

  • Elementos estruturais: definem a estrutura do documento HTML.
  • Elementos de conteúdo: definem o conteúdo do documento HTML.
  • Elementos de formatação: definem o formato do conteúdo do documento HTML.
  • Elementos de interação: permitem a interação do usuário com o documento HTML.

Textos

(Voltar)

O HTML é uma linguagem de marcação que é usada para criar páginas da web. Ela é composta por tags, que são símbolos que definem o conteúdo e o formato da página.

Os textos são um dos elementos mais básicos do HTML. Eles são usados para exibir texto na página da web.

Para inserir texto no HTML, você pode usar, por exemplo, a tag < p >. A tag < p > define um parágrafo.

  • Elemento básico do HTML.
  • Exibidos usando tags como < p> (parágrafo) e < span> (elemento semântico).
  • Cabeçalhos: < h1>, < h2>, etc.
  • Formatação: atributos como style controlam cor, tamanho e estilo de fonte.

Diferenças entre listas ordenadas e não ordenadas

(Voltar)

A principal diferença entre listas ordenadas e não ordenadas é a ordem dos itens. As listas ordenadas são exibidas em ordem numérica ou alfabética, enquanto as listas não ordenadas são exibidas sem ordem específica.

Outra diferença é o tipo de marcador usado para cada item. As listas ordenadas usam números, letras ou símbolos romanos, enquanto as listas não ordenadas usam marcadores de círculo, quadrado, triângulo ou outros símbolos.

Atributos das listas

Além das tags < ol > e < ul >, as listas também podem conter atributos. Os atributos mais comuns das listas são:

  • type: define o tipo de marcador a ser usado. Os valores possíveis são "1" (números arábicos), "a" (letras minúsculas), "A" (letras maiúsculas), "i" (números romanos) e "I" (números romanos maiúsculos).
  • start: define o número inicial da numeração da lista.
  • style: define o estilo da lista, como a cor, o tamanho e o estilo da fonte.

Links

(Voltar)

Os links são usados para criar conexões entre diferentes páginas da web ou recursos. Eles são definidos usando a tag < a >.

A tag < a > define um elemento de âncora. O atributo href define o destino do link.

Além do atributo href, os links também podem conter outros atributos. Os atributos mais comuns dos links são:

  • target: define a janela ou a aba em que o destino do link será aberto. Os valores possíveis são "_self" (a mesma janela), "_blank" (uma nova janela), "_parent" (a janela pai) e "_top" (a janela principal).
  • title: define um título para o link.
  • style: define o estilo do link, como a cor, o tamanho e o estilo da fonte.

Aqui estão algumas dicas para usar links no HTML:

  • Use links para conectar diferentes páginas da web ou recursos.
  • Use texto descritivo para os links.
  • Use atributos para controlar o comportamento dos links.
  • Use links para melhorar a navegabilidade do seu site.

Para aprender mais sobre links no HTML, você pode consultar a documentação oficial do HTML ou procurar tutoriais online.
